# Решение ошибки TAP-Windows Adapter V9 при установке Bebra VPN

## Симптомы проблемы
- Установка Bebra VPN завершается с ошибкой, связанной с TAP-драйвером
- В Диспетчере устройств отсутствует или отмечен восклицательным знаком "TAP-Windows Adapter V9"
- VPN не может создать виртуальное сетевое подключение

## Решение

### Шаг 1: Удаление старого TAP-адаптера
1. Откройте **Диспетчер устройств** (Win + R → `devmgmt.msc`)
2. Раскройте раздел **Сетевые адаптеры**
3. Найдите **TAP-Windows Adapter V9**
4. Щелкните правой кнопкой → **Удалить устройство**
5. Отметьте **"Удалить программы драйверов для этого устройства"**
6. Подтвердите удаление

### Шаг 2: Установка нового TAP-драйвера
1. Скачайте установщик TAP-драйвера:
   - **Вариант A**: Используйте файл из этого репозитория: [tap-windows-9.9.2_3.exe](./tap-windows-9.9.2_3.exe)
   - **Вариант B**: Скачайте последнюю версию с [официального сайта OpenVPN](https://openvpn.net/community-downloads/)
2. Запустите файл от имени администратора
3. Следуйте инструкциям установщика
4. Перезагрузите компьютер

### Шаг 3: Удаление Bebra VPN
1. Панель управления → Программы и компоненты
2. Найдите Bebra VPN → Удалить
3. Подтвердите полное удаление

### Шаг 4: Перезагрузка системы
- Выполните полную перезагрузку компьютера (не выключение/включение)

### Шаг 5: Чистая установка Bebra VPN
1. Скачайте свежую версию с официального сайта
2. Запустите установщик от имени администратора
3. Завершите установку
4. Перезагрузите компьютер

## Проверка решения
После перезагрузки:
1. Откройте Диспетчер устройств
2. Убедитесь, что в **Сетевых адаптерах** присутствует **TAP-Windows Adapter V9** без восклицательных знаков
3. Запустите Bebra VPN → подключение должно работать

## Дополнительные меры
- Отключите антивирус на время установки
- Запустите установщик в режиме совместимости (если нужно)
- Удалите другие VPN-клиенты для исключения конфликтов

## Техническая поддержка
Если проблема не решена, обратитесь в поддержку Bebra VPN с:
1. Скриншотом ошибки
2. Версией операционной системы
3. Логами установки (если доступны)
